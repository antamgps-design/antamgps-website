# An Tâm GPS Platform v2 — Data Build v1

Phạm vi đã triển khai từ dữ liệu hiện có:
- Homepage hoàn chỉnh
- Danh mục sản phẩm
- Trang chi tiết H29P, GS12, S20, EV04
- Trang ứng dụng IOPGPS dùng chung
- Download Center
- Knowledge Center
- QR tải app, QR Zalo và video ứng dụng

## Chạy trên Windows
```cmd
npm.cmd install
npm.cmd run dev
```
Mở http://localhost:4321

Các ảnh đội ngũ, khách hàng và đối tác chưa được đưa vào; có thể bổ sung sau mà không đổi cấu trúc.
